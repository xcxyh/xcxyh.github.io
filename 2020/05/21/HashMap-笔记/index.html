HashMap 的 put 方法

当我们调用put方法时，首先对传入的 key 进行 hash(key) 操作，这是一个扰动函数，具体作用是让key的hashcode的高16位也参与运算。以减少hash冲突的概率。从中也可以看出，key 为null的值默认的存放位置为 0 。

onlyIfAbsent 参数 为true时，当传入的key存在时，不执行插入替换操作，一般为false，有则替换，没有就插入。evict
